WINMERGE

Το WinMerge είναι εργαλείο συγκρίσεως και συγχωνεύσεως, Ελευθέρου Κώδικα,
το οποίο είναι συμβατό με όλες τις σύγχρονες εκδόσεις του Windows.
Ορισμένα χαρακτηριστικά απαιτούν πρόσθετες εγκαταστάσεις ή στοιχεία.

Η τελευταία έκδοση του WinMerge και άλλες πληροφορίες για το WinMerge
είναι διαθέσιμα στη διεύθυνση http://winmerge.org

Γρήγορη εισαγωγή στο WinMerge:
Διαβάστε το κεφάλαιο Γρήγορη Εισαγωγή του εγχειριδίου στο διαδίκτυο (στα
Αγγλικά) για να ξεκινήσετε να χρησιμοποιείτε το WinMerge:
http://www.winmerge.org/2.8/manual/quickstart.html

Εγχειρίδιο HTML:
Το εγχειρίδιο είναι διαθέσιμο στο διαδίκτυο στη διεύθυνση
http://www.winmerge.org/2.8/manual/
είναι επίσης πιθανώς εγκατεστημένο (εάν επελέγη) τοπικά και είναι δυνατόν
να μεταφερθεί ξεχωριστά από τη διεύθυνση http://www.winmerge.org/
(βλ. συνοδευτικό κατατοπιστικό υλικό)

Υποστήριξη Συμπιεσμένων Αρχείων:
Το WinMerge χρησιμοποιεί το 7-zip για την υποστήριξη συμπιεσμένων αρχείων.
Το 7-zip (http://www.7-zip.org) είναι συμπιεστικό/αποσυμπιεστικό εργαλείο,
Ελευθέρου Κώδικα. Για την εγκατάσταση υποστηρίξεως συμπιεσμένων αρχείων,
μεταφέρατε το πακέτο εγκαταστάσεως του αρθρώματος 7-zip από τη διεύθυνση
http://www.winmerge.org/.

Εγκατάσταση Υποστηρίξεως Συμπιεσμένων Αρχείων:
Συνιστάται η εγκατάσταση της εφαρμογής 7-zip. Εάν αυτό δεν είναι δυνατόν
για κάποιο λόγο, το πακέτο εγκαταστάσεως του αρθρώματος 7-zip είναι δυνατόν
να εγκαταστήσει μόνο τα απαιτούμενα αρχεία ώστε να ενεργοποιηθεί η υποστήριξη
συμπιεσμένων αρχείων. Σημειώστε ότι ΔΕΝ ΕΠΙΤΡΕΠΕΙ αυτόνομη χρήση του 7-zip,
μόνο υποστήριξη συμπιεσμένων αρχείων για το WinMerge.

Αρχεία Δέσμης Ενεργειών (Scripts):
Εάν επιθυμείτε να εργάζεσθε με αρχεία δέσμης ενεργειών χρειάζεται να έχετε
εγκατεστημένο το Windows Scripting Host (Διερμηνέας Αρχείων Δέσμης Ενεργειών
του Windows). Εάν σας εμφανισθούν οποιαδήποτε μηνύματα σφαλμάτων σχετικά με
τα αρχεία σας δέσμης ενεργειών παρακαλούμε επισκεφθείτε τη διεύθυνση
http://msdn.microsoft.com/library/default.asp?url=/downloads/list/webdev.asp
ώστε να εξασφαλίσετε ότι ο Scripting Host σας είναι και ενημερωμένος με την
τελευταία έκδοση και ακέραιος.

Προγραμματιστικά Σφάλματα (Bugs) και Αιτήσεις για επί πλέον Δυνατότητες:
Προγραμματιστικά σφάλματα και προτάσεις για νέα χαρακτηριστικά θα πρέπει
να υποβάλλονται στα πρωτόκολα (trackers) σφαλμάτων και αιτήσεων για
χαρακτηριστικά στο sourceforge.net.

Πρωτόκολο Προγραμματιστικών Σφαλμάτων:
http://sourceforge.net/tracker/?group_id=13216&atid=113216
Όταν αναφέρετε ένα προγραμματιστικό σφάλμα, παρακαλούμε πείτε μας για
τον αριθμό εκδόσεως του WinMerge! Οι εκδόσεις του WinMerge 2.2.0 και
μεταγενέστερες έχουν τη δυνατότητα να δημιουργήσουν "Καταγραφή Ρυθμίσεων"
από την επιλογή Βοήθεια/Ρυθμίσεις. Παρακαλούμε επισυνάψατε αυτές τις
πληροφορίες (ως συνημμένο αρχείο) στην αναφορά σας για προγραμματιστικό
σφάλμα, περιέχουν πλήθος χρησίμων πληροφοριών για προγραμματιστές.
